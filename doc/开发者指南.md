# geektime-downloader 项目开发者指南

## 项目概览

### 项目介绍
geektime-downloader 是一个功能完善的极客时间资源下载器，支持下载专栏文章（PDF/Markdown/音频格式）、视频课程、每日一课、大厂案例和训练营视频等多种类型的学习资源。

经过优化后，项目已从单纯的交互式工具演进为支持自动化的企业级下载解决方案。

### 技术栈
- **编程语言**: Go 1.16+
- **核心框架**: 
  - `cobra` - CLI 命令行框架
  - `resty` - HTTP 客户端库
  - `chromedp` - 浏览器自动化（PDF生成）
  - `promptui` - 交互式命令行
  - `yaml` - 配置文件解析

### 项目特色
- 🚀 **多模式支持** - 交互式 + 非交互式 + 配置文件
- 📦 **批量下载** - 一次处理多个课程
- ⚙️ **灵活配置** - 支持全局配置和课程级别覆盖
- 🔧 **自动化友好** - 支持后台运行和CI/CD集成
- 🌍 **跨平台构建** - 一键生成9个平台的可执行文件

### 核心模块

#### 1. 命令行处理 (cmd/)
**职责**: 用户交互、参数解析、模式切换、业务流程编排

**关键特性**:
- 智能模式检测（交互式/非交互式）
- 完整的参数验证和错误处理
- 支持批量处理和配置文件

**主要函数**:
```go
// 模式检测
func shouldRunNonInteractive() bool

// 非交互式主流程
func runNonInteractiveMode(ctx context.Context) error

// 配置文件处理
func runBatchDownloadFromConfig(ctx context.Context) error

// 单课程下载
func downloadCourseByConfig(ctx context.Context, config CourseConfig) error
```

#### 2. API客户端 (internal/geektime/)
**职责**: 封装极客时间网站的所有API调用

**关键特性**:
- 统一的HTTP客户端管理
- 完整的认证和会话管理
- 支持普通版和企业版API

**核心接口**:
```go
// 创建客户端
func NewClient() *Client

// 课程信息
func (c *Client) CourseInfo(productID int, productType int) (*CourseInfo, error)

// 文章详情
func (c *Client) V1ArticleInfo(id int) (*Article, error)
```

#### 3. 下载引擎 (internal/video/, internal/pdf/, etc.)
**职责**: 处理不同类型内容的下载和转换

**关键特性**:
- 并发下载和进度显示
- 多格式支持（PDF/Markdown/音频/视频）
- 断点续传和错误恢复

#### 4. 配置管理
**职责**: 处理复杂的配置继承和覆盖逻辑

**配置优先级**:
```
命令行参数 > 课程配置 > 全局配置 > 程序默认值
```

**配置结构**:
```go
type GlobalConfig struct {
    GCID              string `yaml:"gcid,omitempty"`
    GCESS             string `yaml:"gcess,omitempty"`
    Folder            string `yaml:"folder,omitempty"`
    Quality           string `yaml:"quality,omitempty"`
    Output            int    `yaml:"output,omitempty"`
    // ... 其他配置项

type CourseConfig struct {
    ID          int    `yaml:"id"`
    Type        string `yaml:"type"`
    DownloadAll bool   `yaml:"download_all"`
    ArticleIDs  []int  `yaml:"article_ids,omitempty"`
    // 可选覆盖配置（使用指针实现）
    Quality     *string `yaml:"quality,omitempty"`
    Output      *int    `yaml:"output,omitempty"`
    // ...
}
```

## 关键技术实现

### 1. 智能模式切换
程序会自动检测用户意图并切换到相应模式：

```go
func shouldRunNonInteractive() bool {
    return courseIDs != "" || configFile != ""
}
```

### 2. 批量处理逻辑
支持错误容错的批量处理：

```go
func processBatchCourses(courses []CourseConfig) error {
    for i, course := range courses {
        fmt.Printf("处理课程 %d/%d: %d\n", i+1, len(courses), course.ID)
        
        if err := downloadCourseByConfig(ctx, course); err != nil {
            fmt.Printf("课程 %d 下载失败: %v\n", course.ID, err)
            continue // 继续处理其他课程
        }
    }
    return nil
}
```

### 3. 配置继承机制
实现了灵活的配置覆盖逻辑：

```go
func applyCourseConfig(course CourseConfig, global GlobalConfig) {
    // 只有当参数为默认值时，才应用配置文件设置
    if quality == "sd" && course.Quality != nil {
        quality = *course.Quality
    }
    
    if output == 1 && course.Output != nil {
        output = *course.Output
    }
    // ...
}
```

### 4. 并发下载优化
```go
// 使用errgroup进行并发控制
g, ctx := errgroup.WithContext(ctx)
g.SetLimit(concurrency)

for _, article := range articles {
    article := article
    g.Go(func() error {
        return downloadArticle(ctx, article)
    })
}

return g.Wait()
```

## 故障排除

### 常见问题

1. **认证失败**:
   - 检查cookie是否过期
   - 确认账户状态正常
   - 验证网络连接

2. **下载失败**:
   - 检查磁盘空间
   - 确认文件权限
   - 查看错误日志

3. **编译问题**:
   - 确认Go版本 >= 1.16
   - 检查依赖是否完整
   - 运行 `go mod tidy`
