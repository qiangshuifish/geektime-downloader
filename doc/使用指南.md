# geektime-downloader 使用指南

## 概述

geektime-downloader 是一个功能强大的极客时间资源下载工具，支持多种使用模式，从简单的交互式下载到复杂的自动化批量处理。

## 安装和环境要求

### 系统要求
- 操作系统：Windows, Linux, macOS
- Chrome/Chromium 浏览器（用于PDF生成）

### 安装方式

#### 方式1：下载预编译版本
1. 访问 [Releases页面] 下载对应平台的可执行文件
2. 解压并将文件移至系统PATH目录

#### 方式2：从源码编译
```bash
git clone <repository-url>
cd geektime-downloader
go build -o geektime-downloader
```

#### 方式3：使用构建脚本
```bash
./build.sh                    # 构建所有平台
./build.sh -p linux-amd64     # 构建特定平台
```

## 使用模式概览

geektime-downloader 支持三种使用模式：

| 模式 | 适用场景 | 触发条件 |
|------|----------|----------|
| **交互式模式** | 新手用户、单次下载 | 默认模式（无特殊参数） |
| **命令行模式** | 简单批量、脚本调用 | 指定 `--course-ids` 参数 |
| **配置文件模式** | 复杂批量、自动化 | 指定 `--config` 参数 |

## 模式一：交互式使用

### 基本流程
```bash
# 启动程序
./geektime-downloader

# 根据提示操作：
# 1. 选择产品类型
# 2. 输入课程ID
# 3. 选择下载内容
# 4. 开始下载
```

### 认证方式
- **自动获取**：程序会引导您通过浏览器获取认证信息
- **手动输入**：也可以直接输入 gcid 和 gcess 值

### 支持的产品类型
- **普通专栏** (normal) - 专栏课程
- **每日一课** (daily) - 每日一课
- **公开课** (openclass) - 公开课/视频课
- **大厂案例** (qconplus) - QCon+案例
- **训练营** (university) - 极客大学
- **其他** (other) - 其他类型课程

## 模式二：命令行使用

### 基本语法
```bash
geektime-downloader [全局选项] --course-ids <课程ID> --product-type <类型> [其他选项]
```

### 核心参数

#### 认证参数
```bash
--gcid "your_gcid"           # 极客时间 cookie 值 gcid
--gcess "your_gcess"         # 极客时间 cookie 值 gcess
```

#### 课程参数
```bash
--course-ids "100056701,100056702"  # 课程ID列表（逗号分隔）
--product-type "normal"             # 产品类型
--download-all true                 # 是否下载全部内容（默认true）
--article-ids "1,2,3"               # 指定文章ID（仅当download-all=false时）
```

#### 下载配置
```bash
-f, --folder "/path/to/download"    # 下载目录
-q, --quality "hd"                  # 视频质量：ld/sd/hd
--output 7                          # 输出格式：1(PDF)+2(Markdown)+4(音频)
--comments true                     # 是否下载评论
--interval 2                        # 下载间隔（秒）
--concurrency 6                     # 并发下载数
```

#### PDF配置
```bash
--print-pdf-wait 8                  # PDF生成等待时间（秒）
--print-pdf-timeout 60              # PDF生成超时时间（秒）
```

#### 特殊选项
```bash
--enterprise                        # 企业版极客时间
--non-interactive                   # 强制非交互模式
```

### 使用示例

#### 1. 下载单个普通专栏
```bash
geektime-downloader \
  --gcid "your_gcid" \
  --gcess "your_gcess" \
  --course-ids 100056701 \
  --product-type normal
```

#### 2. 批量下载多个课程
```bash
geektime-downloader \
  --gcid "your_gcid" \
  --gcess "your_gcess" \
  --course-ids 100056701,100056702,100056703 \
  --product-type normal \
  --quality hd \
  --output 7
```

#### 3. 下载指定文章
```bash
geektime-downloader \
  --gcid "your_gcid" \
  --gcess "your_gcess" \
  --course-ids 100056701 \
  --product-type normal \
  --download-all=false \
  --article-ids 1,2,3,5,8
```

#### 4. 高质量完整下载
```bash
geektime-downloader \
  --gcid "your_gcid" \
  --gcess "your_gcess" \
  --course-ids 100056701 \
  --product-type normal \
  --quality hd \
  --output 7 \
  --comments true \
  --folder "/Users/username/GeekTime"
```

## 模式三：配置文件使用

### 配置文件格式

配置文件采用 YAML 格式，支持全局配置和课程级别配置：

```yaml
# 全局配置 - 所有课程的默认设置
global:
  gcid: "your_gcid_here"
  gcess: "your_gcess_here"
  folder: "/Users/username/Downloads/geektime"
  quality: "hd"
  output: 7
  comments: true
  interval: 2
  concurrency: 6
  enterprise: false

# 基础课程配置
courses:
  - id: 100056701
    type: normal
    download_all: true
  
  - id: 100056702
    type: normal
    download_all: false
    article_ids: [1, 2, 3, 5, 8]
  
  - id: 100122405
    type: daily
    download_all: true

# 高级课程配置 - 支持覆盖全局设置
advanced_courses:
  - id: 100056703
    type: normal
    download_all: true
    # 覆盖全局设置
    folder: "/special/path/for/this/course"
    quality: "ld"
    output: 2
    comments: false
    interval: 5
```

### 配置项说明

#### 全局配置 (global)
| 配置项 | 类型 | 必需 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `gcid` | string | 是* | - | 极客时间 cookie 值 |
| `gcess` | string | 是* | - | 极客时间 cookie 值 |
| `folder` | string | 否 | ~/geektime-downloader | 下载目录 |
| `quality` | string | 否 | sd | 视频质量: ld/sd/hd |
| `output` | int | 否 | 1 | 输出格式: 1(PDF)/2(Markdown)/4(音频) |
| `comments` | bool | 否 | true | 是否下载评论 |
| `interval` | int | 否 | 1 | 下载间隔(秒) |
| `concurrency` | int | 否 | 自动 | 并发下载数 |
| `print_pdf_wait` | int | 否 | 8 | PDF生成等待时间(秒) |
| `print_pdf_timeout` | int | 否 | 60 | PDF生成超时时间(秒) |
| `enterprise` | bool | 否 | false | 是否为企业版 |

*注：认证信息也可通过命令行参数提供

#### 课程配置
| 配置项 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| `id` | int | 是 | 课程ID |
| `type` | string | 是 | 课程类型 |
| `download_all` | bool | 是 | 是否下载全部 |
| `article_ids` | []int | 否 | 指定文章ID列表 |

**可选覆盖配置**（支持覆盖任意全局配置）：
- `quality` - 视频质量
- `output` - 输出格式  
- `comments` - 评论下载
- `folder` - 下载目录
- `interval` - 下载间隔
- `concurrency` - 并发数
- `print_pdf_wait` - PDF等待时间
- `print_pdf_timeout` - PDF超时时间
- `enterprise` - 企业版设置

### 配置文件使用

#### 1. 创建配置文件
```bash
# 复制示例配置
cp courses-example.yaml my-courses.yaml

# 编辑配置文件
vim my-courses.yaml
```

#### 2. 执行下载
```bash
# 使用配置文件（认证信息在配置文件中）
geektime-downloader --config my-courses.yaml

# 命令行提供认证信息（覆盖配置文件）
geektime-downloader --config my-courses.yaml --gcid "xxx" --gcess "yyy"
```

### 配置优先级

配置的应用优先级（从高到低）：

1. **命令行参数** - 最高优先级
2. **课程级别配置** - 覆盖全局配置
3. **全局配置** - 默认设置
4. **程序默认值** - 最低优先级

示例：
```bash
# 命令行的 --quality 会覆盖配置文件中的所有 quality 设置
geektime-downloader --config my-courses.yaml --quality ld
```

## 后台运行和自动化

### 后台运行

#### Linux/macOS
```bash
# 后台运行，输出重定向到文件
nohup geektime-downloader --config courses.yaml > download.log 2>&1 &

# 查看日志
tail -f download.log

# 查看进程状态
ps aux | grep geektime-downloader
```

#### Windows PowerShell
```powershell
# 后台运行
Start-Process geektime-downloader.exe `
  -ArgumentList "--config","courses.yaml" `
  -RedirectStandardOutput "download.log" `
  -RedirectStandardError "error.log" `
  -WindowStyle Hidden

# 查看日志
Get-Content download.log -Wait
```

### 自动化脚本

#### Bash 脚本示例
```bash
#!/bin/bash

# 设置变量
CONFIG_FILE="courses.yaml"
LOG_FILE="download_$(date +%Y%m%d_%H%M%S).log"

# 执行下载
echo "开始批量下载，日志文件: $LOG_FILE"
geektime-downloader --config "$CONFIG_FILE" > "$LOG_FILE" 2>&1

# 检查结果
if [ $? -eq 0 ]; then
    echo "下载完成成功"
else
    echo "下载过程中出现错误，请查看日志: $LOG_FILE"
fi
```

#### 定时任务 (cron)
```bash
# 每天凌晨2点执行下载
0 2 * * * /path/to/geektime-downloader --config /path/to/courses.yaml >> /path/to/cron.log 2>&1
```

### CI/CD 集成

#### GitHub Actions 示例
```yaml
name: Download GeekTime Courses
on:
  schedule:
    - cron: '0 2 * * 0'  # 每周日凌晨2点

jobs:
  download:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Download courses
        run: |
          chmod +x geektime-downloader
          ./geektime-downloader --config courses.yaml
        env:
          GCID: ${{ secrets.GCID }}
          GCESS: ${{ secrets.GCESS }}
          
      - name: Upload artifacts
        uses: actions/upload-artifact@v3
        with:
          name: downloaded-courses
          path: downloads/
```

## 错误处理和故障排除

### 常见错误

#### 1. 认证相关错误
```
错误: argument 'phone' or cookie value is not valid
```
**解决方案**:
- 检查 gcid 和 gcess 是否正确
- 确认 cookie 未过期
- 重新登录获取新的 cookie 值

#### 2. 网络相关错误
```
错误: 网络请求超时
```
**解决方案**:
- 检查网络连接
- 增加超时时间参数
- 降低并发数减少网络压力

#### 3. 文件权限错误
```
错误: 无法创建目录或文件
```
**解决方案**:
- 检查下载目录的写入权限
- 确保磁盘空间充足
- 使用绝对路径避免路径问题

#### 4. 课程访问错误
```
错误: 课程不存在或无权限访问
```
**解决方案**:
- 确认课程ID正确
- 检查是否已购买该课程
- 验证产品类型是否匹配

## 性能优化建议

### 下载性能优化

1. **合理设置并发数**:
   ```yaml
   global:
     concurrency: 4  # 根据网络情况调整
   ```

2. **控制下载间隔**:
   ```yaml
   global:
     interval: 2  # 避免请求过于频繁
   ```

3. **选择合适的质量**:
   ```yaml
   global:
     quality: "sd"  # 标清可以大幅减少下载时间
   ```

### 存储优化

1. **选择性下载格式**:
   ```yaml
   global:
     output: 2  # 只下载Markdown，节省空间
   ```

2. **跳过评论**:
   ```yaml
   global:
     comments: false  # 跳过评论下载
   ```

3. **分批处理**:
   - 对于大量课程，建议分批次下载
   - 避免一次性下载过多内容

### 系统资源优化

1. **内存管理**:
   - 大文件下载时适当降低并发数
   - 定期重启长时间运行的下载任务

2. **磁盘IO优化**:
   - 使用SSD存储提高写入速度
   - 确保足够的临时空间用于文件处理
